# Manoir d'Allinges

Site de vente d'un magnifique manoir du XIXème siècle situé à Allinges, proche de Genève et du Lac Léman.

## Caractéristiques
- 400 m² habitables
- 10 pièces
- 7000 m² de terrain arboré
- Prix : 1 397 000 €
